메인 페이지 중앙에 영상이 자동재생 된다. (비디오 태그 사용, 자동재생, 반복, 음소거)
메인 페이지에서 사진에 마우스 커서를 올리면 글자가 줄어든다. (애니메이션 효과 사용)
실시간 주가 페이지에서 네이버 주식 사이트를 노출했다. (아이프레임 태그 사용)
옵션 선택 페이지에서 설문조사 영역의 모서리를 둥굴게 했다. (보더 레디우스 태그 사용)
고객 정보 입력 페이지에서 성함, 전화번호, 이메일을 입력하는 창에 힌트를 노출했다. (플레이스 홀더 태그 사용)
옵션 선택, 고객 정보 입력 페이지에서 설문조사 영역에 마우스를 올리면 배경색이 옵션 선택에서 하이퍼 링크에 커서를 올리면 폰트 색상이 변하도록 했다. (태그명 : hover 사용)
모든 페이지에 "지마켓 산스 볼드체"를 사용했다. (찾아오시는 길 페이지 제외)
